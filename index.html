Race Around the Kingdom

By Matt Fritz


This game has been changed from its original format. Originally, it was built on October 28, 2010 as a one-day class assignment to demonstrate the Google Maps API while I was at UC Irvine getting my undergraduate degree (Bachelor's of Science in Informatics with a specialty in Human-Computer Interaction). It involved taking latitude/longitude data from a data source and placing it on a map, so I figured I could turn it into a game by using venues from Foursquare. People in the class really seemed to enjoy it and it was voted second place out of eight projects in the class competition.

On December 27, 2011 I had already completed my first semester of grad school (Master's of Science in Software Engineering). I decided I wanted to improve the general functionality of the game by making it totally dependent on the client browser instead of the original method of using server communication for getting questions and checking answers. Over the course of a week I added some other fun things to it too. It now has almost twice the amount of locations to visit and picks location/question combinations randomly.

If you would like to play the game, click the Start Game button above the map and the instructions will automatically be displayed next to the map. You can also click the Show/Hide Rules button to read the game instructions beforehand. Some of the questions are not G-rated and a few are even probably PG-13 (it was, after all, originally a college class assignment). You may also use the map as a regular Google Maps interface to explore the Disneyland Resort from an aerial view on your own.

Download the source if you're inclined: ratk_src.zip (58 KB)

Enjoy!

Game Rules

A description of the game and its controls

Race Around the Kingdom is a relatively straightforward game by Matt Fritz that uses AJAX, JSON, the Google Maps API, jQuery, and other JavaScript to create the product that you are seeing.

The standard UI controls for the map become disabled and disappear after you start the game in order to prevent certain kinds of cheating. During the game, you'll be experiencing Disneyland through satellite imagery and going to different locations in the Resort as you answer questions.

Gameplay Instructions

  1. Click the "Start Game" button to begin the game
  2. One location will reveal itself each time as a marker on the map
  3. When you arrive at the location, the marker's image will change to a star to signal your arrival.
  4. Click the marker to load the next random question
  5. Choose an answer and press "Submit"
  6. If your answer was correct, you'll get a point. If not, your score will be unchanged.
  7. Click "Go to Next Location" to be taken to the next location and continue to experience the Resort!
You can tell how many questions you have left by looking at the number in the bold font when the bubble appears after you click a marker. In total, there are 90 locations to see and 51 questions to answer but you will only see 20 random ones per game.

You can press the "Center Map" button if you need to jump back to the current location to answer its question.

You may also exit the game and return to the normal map format by clicking the "Exit Game" button above the map at any time.

Enjoy!
